# Analysis_ready Data

## What belongs here:
The final, enriched versions of the dataset (e.g. `v1.1.0` with derived columns like `maturation_days` or `v2.0.0` with filtered rows).

## Who should modify files in this folder:
Data Scientists and Analysts.

## When data is allowed to move to the next stage:
This is the final stage; data from here is used directly.

## What never goes into this folder:
Intermediate working files that are not final.
