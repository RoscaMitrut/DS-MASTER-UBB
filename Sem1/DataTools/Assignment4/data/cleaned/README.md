# Cleaned Data

## What belongs here:
Data that has gone through a cleaning process. This includes files where numeric errors have been corrected (e.g.`v1.0.1`).

## Who should modify files in this folder:
Data Scientists performing data cleaning.

## When data is allowed to move to the next stage:
Anytime.

## What never goes into this folder:
The original raw files or final versions as well as files containing experimentally calculated columns.
